## Sol
```xml
피보나치 수는 F(0) = 0, F(1) = 1일 때, 1 이상의 n에 대하여 F(n) = F(n-1) + F(n-2) 가 적용되는 수 입니다.
예를들어
* F(2) = F(0) + F(1) = 0 + 1 = 1
* F(3) = F(1) + F(2) = 1 + 1 = 2
* F(4) = F(2) + F(3) = 1 + 2 = 3
* F(5) = F(3) + F(4) = 2 + 3 = 5 와 같이 이어집니다.  
```
레벨2여서 도전한 문제였지만, 그대로 구현하면 되는 쉬운 문제였다.


## 답안
```python
def solution(n):
    d = [0]*(n+1)
    d[0],d[1] = 0,1
    
    if n < 2: return d[n]

    for i in range(2,n+1):
        d[i] = d[i-1]+d[i-2]
        
    return (d[n] % 1234567)
```
