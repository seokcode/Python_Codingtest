## Sol


수학 문제에 조금 더 가까운 듯 한 문제였다..


정사각형일 경우 대각선에 의해 사라지는 사각형이 가로길이(세로길이)와 같고,  
직사각형일 경우 가로길이+세로길이에서 가로와 세로의 최대공약수만큼을 뺀 것과 같다.


## 답안
```python
def solution(w, h):
    if w == h: return w * h - w

    if w < h:
        w, h = h, w

    def gcd(w, h):
        if h == 0:
            return w
        else:
            return gcd(h, w % h)

    return w * h - (w + h - gcd(w, h))
```
